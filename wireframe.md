# iLearn FCU 首頁 Wireframe

根據目前截圖與 sitemap 整理，以下以「手機版首頁」為主做 wireframe 拆解。

## 1. 頁面骨架

```text
┌──────────────────────────────────────────────┐
│ Header                                       │
│ [漢堡選單]                      [語言] [登入] │
├──────────────────────────────────────────────┤
│ Hero / Banner Carousel                       │
│ ┌──────────────────────────────────────────┐ │
│ │ 主視覺 Banner 圖 / 宣傳文案             │ │
│ │ 右側切換箭頭                             │ │
│ └──────────────────────────────────────────┘ │
│              o  o  o                         │
├──────────────────────────────────────────────┤
│ Spacer / 留白或次要資訊區                    │
│                                              │
├──────────────────────────────────────────────┤
│ FAQ Section                                  │
│                 常見問題                     │
│ ┌──────────────────────────────────────────┐ │
│ │ 1. 問題標題                               │ │
│ └──────────────────────────────────────────┘ │
│ ┌──────────────────────────────────────────┐ │
│ │ 2. 問題標題                               │ │
│ └──────────────────────────────────────────┘ │
├──────────────────────────────────────────────┤
│ Floating Action Button                       │
│                                [電話 / 聯絡]  │
└──────────────────────────────────────────────┘
```

## 2. 元件拆解

### Header
- 漢堡選單按鈕
- 語言切換按鈕 / 下拉
- 登入按鈕
- 白底頂欄

### Hero Banner
- 輪播主視覺容器
- Banner 圖片
- 宣傳文案區
- 左右切換控制
- 輪播頁點指示器

### 中段留白區
- 區塊分隔空間
- 可延伸放活動資訊、公告、捷徑或功能卡片

### FAQ 區塊
- 區塊標題
- FAQ 項目 1
- FAQ 項目 2
- 可展開 / 收合的手風琴元件

### 浮動按鈕
- 聯絡電話 / 客服快捷入口
- 固定在右下角

## 3. 根據 sitemap 補上的延伸元件

這些元件不一定出現在截圖中，但從 sitemap 可推測首頁應該還會連到：
- 學習資源 / 課程導覽
- 教師篇
- 學生篇
- 新進教師工作坊
- 操作指引
- 教務資料下載
- 校園智慧財產權宣導
- Q&A 彙整
- 聯絡我們
- 取得行動應用程式
- 智慧財產權政策

## 4. 建議的 wireframe 分層

1. Global Header
2. Hero Carousel
3. Information Spacer / Content Slot
4. FAQ Accordion List
5. Floating Contact Button
6. Footer / Secondary Links
